# Project Wattpad 📚

> **Notice:** This is a student school project created strictly for educational purposes as part of a front-end development activity.

While the project is inspired by the platform, the user interface layout and all custom visual elements were entirely designed and created by me. The only external asset used is the official logo.

---

## 📌 Project Features & Design


* **Original UI Layout:** The interface design and structure were custom-built from scratch to create a unique reading/browsing experience.
* **Custom Visuals:** All graphics and visual assets (excluding the logo) are original creations developed specifically for this project.
* **Fully Responsive:** Unlike standard desktop-only practices, this project features a fully responsive layout optimized for mobile, tablet, and desktop screens.
* **Static Front-End:** Built purely as a responsive layout exercise; no backend functionality, database, or live user authentication is implemented.

---

## 🛠️ Tech Stack

* **HTML5** - Semantic structure and content setup
* **CSS3** - Custom UI design, layout styling, and media queries for mobile responsiveness
